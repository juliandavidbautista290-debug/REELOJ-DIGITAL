# Reloj Digital

Este proyecto es un **reloj digital en tiempo real** hecho con **HTML, CSS y JavaScript puro**.  
Muestra horas, minutos y segundos actualizándose automáticamente cada segundo.

## Características

- Muestra la **hora actual** de tu sistema.
- Formato de hora **HH:MM:SS**.
- Fondo negro con números color **agua marina**.
- Totalmente responsivo y centrado en la pantalla.
- Hecho completamente en **frontend**, sin necesidad de backend.

## Archivos del proyecto

- `index.html` → archivo principal del reloj.
- `styles.css` → estilos del reloj y la página.
- `script.js` → lógica del reloj que actualiza la hora en tiempo real.
- `README.md` → documentación del proyecto.

## Cómo usarlo

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. ¡Disfruta de tu reloj digital en funcionamiento!
